# American Mortgage Website

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/YOUR_USERNAME/american-mortgage)

## One-Click Deploy Instructions

### Option 1: Deploy to Netlify (Easiest)

1. **Create a GitHub account** (if you don't have one): https://github.com/signup

2. **Click the "Deploy to Netlify" button above** (after uploading this to GitHub)

3. **That's it!** Netlify will:
   - Clone the repo to your GitHub
   - Build and deploy the site
   - Give you a live URL

### Option 2: Drag & Drop Deploy

1. Go to https://app.netlify.com/drop
2. Drag the entire unzipped folder onto the page
3. Your site is live instantly!

**Note:** Drag & drop won't include the CMS admin. For the full CMS, use Option 1.

---

## After Deployment: Enable the CMS Admin

1. In Netlify dashboard, go to **Site settings** → **Identity** → **Enable Identity**
2. Go to **Identity** → **Services** → **Enable Git Gateway**
3. Go to **Identity** → **Invite users** → Enter your email
4. Check your email and accept the invite
5. Visit `yoursite.netlify.app/admin/` to access the CMS

## What You Can Edit

- Site Settings (name, phone, email, address)
- Navigation menu items
- Hero section (titles, buttons, ratings)
- Stats bar numbers
- All 3 Featured Loan cards
- Down Payment Assistance section
- Tools & Calculators
- Agent section
- App section

All changes auto-save and auto-deploy!
